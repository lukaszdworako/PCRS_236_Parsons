Bolded text:
<b>text here</b>

Italicized text:
<i>text here</i>

Underlined text:
<u>text here</u>

Crossed out text:
<del>text here</del>

Modify the size and/or color of text:
<font size='24' color='red'>text here</font>

Insert image:
<img src='url-to-image'>

Insert image and set size:
<img src='url-to-image' width='560' height='315'>

Insert YouTube video:
* While at a video, click on "share" then "embed", and copy the code given. For example:
<iframe width='560' height='315' src='url-to-video' frameborder='0' allowfullscreen></iframe>
